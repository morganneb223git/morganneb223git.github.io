Project 1: PacMen Exercise

I extended my previous work on creating and animating a single Pacman character to dynamically generate multiple Pacmen on demand. By clicking the "Add PacMan" button, users could spawn new Pacmen at random positions within the webpage. Subsequently, by initiating the "Start Game" button, these Pacmen began moving autonomously across the screen at random velocities, with additional logic implemented to ensure they remained within the boundaries of the webpage.

Project 2: Eye Exercise

I implemented JavaScript mouse events to create a dynamic interaction where an "eye" element on a webpage follows the mouse cursor. By leveraging events like "mousemove," I extended this functionality to include a second "eye" element, ensuring both accurately tracked the mouse's position on the screen.

Project 3: Real-Time Bus Tracking

In this activity, I utilized the mapboxgl JavaScript library to enhance maps with customizations, such as adding markers at specific locations. By leveraging MBTA bus data, I animated a marker along bus routes between Harvard and MIT. The process involved obtaining a Mapbox access token, incorporating it into the code, and executing the move() function to initiate the marker animation, ultimately displaying the marker's movement between bus stops.
